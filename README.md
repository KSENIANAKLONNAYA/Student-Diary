# 📚 Student Diary

Мобильное приложение для студентов — расписание, задачи, баллы, флэш-карточки и достижения.

---

## 📱 Возможности

- **Расписание** — создание пар с указанием Ф.И.О. преподавателя, номера аудитории и цветовой маркировки. Поддержка циклических недель: чёт/нечет, 3 или 4 недели.

- **Задачи** — добавление задач с описанием, приоритетом и дедлайном. Возможность прикреплять фото и файлы. Напоминания о сроках.

- **Баллы** — учёт успеваемости по предметам. Бонусная система до 10 дополнительных баллов. Автоматический подсчёт прогресса в процентах.

- **Флэш-карточки** — создание карточек для запоминания учебного материала. Ошибочные карточки автоматически попадают в отдельную колоду для повторения.

- **Достижения** — система мотивации с отслеживанием прогресса. Награды за выполнение задач, серии и особые условия.

- **Аутентификация** — регистрация и вход через сервер с безопасным хранением данных.

---

## 🛠 Технологии

### iOS приложение
- SwiftUI — современный декларативный интерфейс
- UserNotifications — локальные уведомления
- QuickLook — просмотр PDF и изображений
- REST API — синхронизация с сервером
- JSON — кеширование данных на устройстве
---

## 🚀 Установка и запуск

### iOS приложение

Клонируйте репозиторий, откройте проект в Xcode и запустите на симуляторе или устройстве.

### Сервер

Установите зависимости Python, создайте виртуальное окружение и запустите сервер. По умолчанию сервер работает на порту 8000.
